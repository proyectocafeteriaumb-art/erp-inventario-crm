# 5️⃣ Documentación de Trazabilidad

Se mantiene trazabilidad completa entre:

- Historia de Usuario
- Issue en GitHub
- Rama feature
- Commits
- Pull Request
- Merge a dev/main

## Ejemplo de trazabilidad

Historia: HU-03 Visualización de Stock  
Issue: #15  
Rama: feature/stock-live  
PR: #21
